# 官网上线与搜索收录清单

## 上线前

1. 将正式 Logo 覆盖到 `assets/images/` 中对应文件；当前版本使用临时可显示图形。
2. 检查企业微信二维码是否仍有效。
3. 检查公司地址、邮箱和统一社会信用代码。
4. 如拥有完整项目数据库，用原数据替换 `assets/data/catalog.json`，同时更新 `summary.json`。

## 上线后

1. 打开首页、服务中心、项目查询和联系我们页面，确认手机端与电脑端均可访问。
2. 访问 `/robots.txt` 和 `/sitemap.xml`，确认公开可访问。
3. 在百度搜索资源平台验证网站并提交 sitemap。
4. 在 Bing Webmaster Tools 和 Google Search Console 验证网站并提交 sitemap。
5. 公众号、视频号、小红书、知乎、企查查等平台统一填写公司全称、品牌名和官网。
6. 后续持续发布专业问答、服务指南和合规的匿名化应用场景。
