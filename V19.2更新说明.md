- 100 个专业分类、5 个业务板块

- 公司介绍、服务、案例和专题指南页面

- 移动端固定“项目查询 / 计算模拟 / 采购测试”咨询栏

## 上传 GitHub Pages

1. 将本文件夹内的所有文件上传到仓库根目录，不能只上传外层文件夹。
2. 仓库 Settings → Pages。
3. Source 选择 `Deploy from a branch`。
4. Branch 选择 `main`，目录选择 `/ (root)`，保存。
5. 保留 `.nojekyll` 和 `CNAME`。当前自定义域名为 `www.hongqitengda.com`。
6. GitHub 部署完成后，检查：
   - `/index.html`
   - `/catalog.html`
   - `/category/index.html`
   - `/project/sim-0001.html` 或其他实际项目页
   - `/sitemap.xml`

## 重要说明

- 本站仍为纯静态网站，无数据库、无服务器、无登录后台。
- `catalog.json` 仍保留并用于浏览器端站内检索。
- 静态详情页是额外生成的搜索引擎入口，不替代或删除原检索功能。
- 页面价格、规格和周期均为参考信息，以最终确认内容为准。
