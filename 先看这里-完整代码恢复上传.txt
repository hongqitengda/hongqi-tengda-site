红祺腾达官网完整代码恢复版（不包含 .git）

本压缩包包含完整网站代码、1612 个项目数据、1612 个项目详情页和 1612 张独立项目图片。

使用方法：
1. 在 GitHub Desktop 当前冲突窗口中先选择 Abort merge（如果仍有冲突窗口）。
2. Repository → Show in Explorer，打开本地仓库根目录。
3. 根目录保留原有 .git 文件夹；不要复制或覆盖其他来源的 .git。
4. 将本压缩包解压后，本文件所在目录中的全部内容复制到仓库根目录。
5. 正确状态：.git、index.html、CNAME、assets、project、category 等处在同一级。
6. 回到 GitHub Desktop，确认 Changes 中出现完整新增/修改内容。
7. Commit to main，然后 Push origin。
8. GitHub 仓库 Settings → Pages：main / (root)，自定义域名为 www.hongqitengda.com。

重要：不要点击 Force push；不要将压缩包最外层文件夹再套进仓库根目录。
